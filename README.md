# Ultrasonic-Radar-System
Ultrasonic Radar System for Object Detection
# Project Overview

The Ultrasonic Radar System is an Arduino-based project designed to detect objects and measure their distance in real time. The system uses an ultrasonic sensor mounted on a servo motor to scan the surrounding area, similar to a radar system.

This project helps in visualizing object detection and distance measurement using sensor technology.

# Objectives

Detect nearby objects using ultrasonic sensors

Measure distance accurately

Display object position during radar scanning

# Technologies Used

# Working Process

The ultrasonic sensor sends sound waves.

The waves hit an object and return as an echo.

The sensor measures the time taken for the echo.

Distance is calculated using the speed of sound.

The servo motor rotates to scan different angles.
